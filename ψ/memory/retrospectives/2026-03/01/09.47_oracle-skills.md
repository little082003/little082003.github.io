## Lessons Learned
1. เมื่อทำการติดตั้ง executable utilities ลง Windows แล้วจะเรียกใช้งานต่อ ห้ามพึ่งพาการทำงานอัตโนมัติของ `$env:PATH` แต่ควรประกอบพาธ์เต็ม (Absolute path) มาเรียกใช้แทนเสมอ
2. Command arguments แบบ string Array (เช่น flags ที่ให้รับค่าหลายตัวพร้อมกันด้วยเครื่องหมายจุลภาค) มักจะมีขอบเขตปัญหาการตีความของ CLI ดังนั้นการแตกคำสั่งเป็นหลายบรรทัดแยกเป้าหมายให้ชัดเจน จะเป็นวิธีที่ไว้ใจได้มากที่สุด
3. การอ่าน Context ของ Log ควรอ้างอิงการข้ามแบบมีเงื่อนไข (Conditional Fail-Safe) ในกรณีที่มันไม่ได้สำคัญเชิงโครงสร้าง เพื่อลดจำนวน Tool Errors ลง
